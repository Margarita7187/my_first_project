1. Основное свойство CSS Grid - display: grid. Отличие от Flexbox в том, что Grid работает с двумерными макетами (строки и колонки одновременно), а Flexbox - с одномерными (либо строка, либо колонка).

2. grid-template-columns явно задает размеры и количество колонок, а grid-auto-columns определяет размер автоматически создаваемых колонок, когда элементов больше чем задано в шаблоне. grid-template-columns используется для явного задания сетки, grid-auto-columns - для автоматического дополнения.

3. repeat() повторяет шаблон колонок/строк (repeat(3, 1fr) - 3 колонки по 1fr), minmax() задает минимальный и максимальный размер (minmax(200px, 1fr)), auto-fit автоматически размещает колонки заполняя доступное пространство. Пример: grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)).

4. Именованные grid-области позволяют задавать макет с помощью grid-template-areas, присваивая элементам имена областей. Преимущества: наглядность структуры, простое изменение макета без переопределения позиций элементов, удобство для адаптивного дизайна.

5. Grid-линии нумеруются от 1 до n+1, где n - количество треков. grid-column: 1/3 размещает элемент от первой до третьей линии (занимает 2 колонки), grid-row: 2/4 - от второй до четвертой линии (2 строки). Можно использовать span для указания количества занимаемых треков.

6. grid-gap (или gap) создает равные промежутки между всеми элементами сетки, автоматически применяясь только между элементами. Лучше margins тем, что не требует сложных вычислений, не влияет на внешние отступы контейнера и дает единообразные расстояния.

7. justify-items выравнивает содержимое элементов по горизонтали внутри их ячеек, align-items - по вертикали. Они работают внутри каждой ячейки сетки, в отличие от justify-content/align-content которые выравнивают саму сетку внутри контейнера.

8. auto-fill создает максимально возможное количество колонок заданного размера, даже пустых, а auto-fit растягивает существующие колонки чтобы заполнить все доступное пространство. Auto-fill оставляет пустое место, auto-fit его заполняет.

9. Сложные макеты создаются через grid-template-areas с именами областей в кавычках. Правила: каждая строка в отдельных кавычках, одинаковое количество областей в каждой строке, точка означает пустую ячейку. Элементам задается grid-area с соответствующим именем.

10. CSS Grid позволяет создавать сложные двумерные макеты с точным позиционированием, легко управлять выравниванием и распределением пространства, создавать адаптивные сетки без медиа-запросов, работать с наслаиванием элементов и сложными формами layouts.
