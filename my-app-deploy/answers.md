## Ответы на вопросы для самоконтроля

1. **Основные различия между development и production:**
   - Development: подробные логи, горячая перезагрузка, отладочная информация
   - Production: минимальные логи, оптимизация, безопасность, сжатие кода

2. **Важность environment variables:**
   - Безопасность конфиденциальных данных (ключи, пароли)
   - Разные конфигурации для разных сред
   - Легкость развертывания без изменения кода

3. **Управление разными средами:**
   - Использование .env файлов для каждой среды
   - Флаги в коде по значению NODE_ENV
   - Разные базы данных и внешние сервисы

4. **Методы оптимизации производительности:**
   - Минификация и сжатие кода
   - Кэширование статических ресурсов
   - CDN для распространения контента
   - Оптимизация изображений

5. **Настройка мониторинга ошибок:**
   - Сервисы типа Sentry, LogRocket
   - Логирование на сервере
   - Health checks и метрики
   - Оповещения при ошибках

6. **CI/CD преимущества:**
   - Автоматизация тестирования и деплоя
   - Быстрое и надежное развертывание
   - Минимизация человеческих ошибок
   - Непрерывная доставка обновлений

7. **Обновление без downtime:**
   - Blue-green deployment
   - Canary releases
   - Load balancer с health checks
   - Миграции базы данных без блокировок

8. **Меры безопасности при деплое:**
   - HTTPS для всех соединений
   - Защита environment variables
   - Регулярное обновление зависимостей
   - Security headers (CSP, HSTS)
   - Валидация и санитизация входных данных
